# Itara — World's First All-in-One AI Marketplace

> Buy. Sell. Build the Future of AI.

Official launch page for [itara.ai](https://itara.ai) — the world's first all-in-one AI commerce platform combining GPU rentals, an AI asset marketplace, and AI-skilled freelancers under one roof.

## Stack

- Pure HTML / CSS / JS — zero dependencies, zero build step
- Fonts: Cabinet Grotesk + Satoshi (Fontshare), JetBrains Mono (Google Fonts)
- Deployed on Vercel, custom domain: itara.ai

## Structure

```
itara-launch/
├── index.html      # Main launch page
├── style.css       # Full design system + white/dark theme
├── script.js       # Waitlist form, scroll animations, theme toggle
└── README.md
```

## Local Development

Just open `index.html` in your browser — no build step required.

## Deployment

Connected to Vercel via this GitHub repo. Every push to `main` auto-deploys to [itara.ai](https://itara.ai).

## V1 Modules

1. **GPU Rentals** — Aggregated compute from RunPod, Vast.ai, Lambda Labs
2. **AI Marketplace** — Buy/sell/auction AI agents, models, datasets, hardware
3. **AI Task Freelancer** — Post AI tasks, hire AI-skilled talent

All transactions are escrow-protected.
